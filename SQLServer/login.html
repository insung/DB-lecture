SQL Server 운영 기초

로그인 계정

  • 새로운 계정을 생성하는 방법과 그 계정으로 SQL Server 에 접속하는 방법에 대해 알아보자.

로그인 계정 생성

  1. 데이터베이스 > 보안 > 로그인 > 새 로그인
  2. 이름 입력, SQL Server 인증 선택, 암호 정책 강제 적용 해제
  3. 서버 역할에 체크
  4. 사용자 매핑 > 데이터베이스 역할 멤버에 체크

데이터베이스 역할 멤버

Let's Prcactice

  1. 설치 시, 윈도우 인증으로 설치 했던 PC 에 신규 생성한 로그인 계정으로 접속이 될까?
  2. 최신 SQL Server 는 sa 계정을 허용하지 않는다. 그 이유는 무엇일까?
  3. 암호 정책 강제 적용을 체크 해서 사용해 보자.
  4. 서버 역할에서 sysadmin 역할이 무엇인지 알아보자.
