SQL Server 운영 기초

Port 변경

  • SQL Server 기본 포트는 1433 이다. 포트 변경하는 방법에 대해 알아보자.
  1. 시작 > SQL Server Configuration Manager
  2. SQL Server 네트워크 구성 > DB 인스턴스 > TCP/IP 속성
  3. IP ALL > TCP 포트에 변경할 포트 입력
