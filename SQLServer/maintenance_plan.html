유지 관리 계획 마법사

  • 유지 관리 계획 마법사에서 어떤 작업을 할 수 있는지 알아보자.
  • 유지 관리 계획 마법사로 간단하게 전체 백업을 구성하는 방법에 대해 알아보자.

유지 관리 계획 마법사 구성 방법

  1. 관리 > 유지 관리 계획 > 유지 관리 계획 마법사
  2. 전체 백업은 매월 15일 오전 4시에 수행되도록 설정 후 확인
  3. SQL Server 에이전트 > 작업 > 새 작업
  4. 데이터베이스 백업(전체) 선택 후 다음
  5. 일반 > 데이터베이스 > test 데이터베이스 선택
  6. 대상 > "c:\temp" 으로 변경 후 확인
  7. 보고서 옵션에서 전자 메일 선택 후 다음
  8. 마침
  9. 생성된 유지 관리 계획 선택 > 실행
  10. 실행 완료
