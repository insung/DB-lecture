설치 및 설정 변경

설치

  • SQL Server 설치하는 방법에 대해 알아보자.
  • SQL Server 2017 Express 설치 (링크)
  • SQL Server Management Studio 17.9 설치 (링크)

신규 설치 방법

  1. 새 SQL Server 설치
  2. 기능 선택
  3. 인스턴스 구성
  4. 서버 구성 -> 서비스 계정
  5. 서버 구성 -> 인증 모드
  6. SSMS (SQL Server Management Studio) 설치

Let's Prcactice

  3. 5번 항목에서 혼합 모드 선택하여 설치해 보자.
