라이선스 정책 (license_fields, license_fields_locales, license_values)


※ 테이블 설명

  • type_e column 의 정보는 enums 테이블에서 다음 쿼리로 확인할 수 있다.
    SELECT *
FROM enums
WHERE type = 'type'
  • license_fields_locales 테이블은 license_fields 의 한국어 지원을 위해 존재하는 테이블이다.
  • 다음 쿼리로 1:1 매핑됨을 알 수 있다.
    SELECT fields.id, 
       fields.name, 
       fields.description, 
       fields.section, 
       locales.description_translation, 
       section_translation
FROM license_fields AS fields
INNER JOIN license_fields_locales AS locales
      ON fields.id = locales.license_field_id

※ license_values 테이블 설명

  • license_values 테이블은 라이선스 정책의 전체/부서/사용자의 라이선스 정책 값이 저장되는 테이블이다.
  • scope_e 가 0이면 전체, 1이면 부서, 2이면 사용자이다. scope_id 는 부서 또는 사용자의 id 이다. (참고로 scope_e 가 0 이면 scope_id 도 0 이다.)
