설치 방법

  1. PostgreSQL 9.5 설치
  2. postgres 계정의 암호 입력
  3. 포트 입력
  4. Stack Builder 체크 해제 후 마침

접속 방법

  1. 시작 > PostgreSQL 9.5 > pgAdmin3 > PostgreSQL 9.5 더블 클릭
  2. 비밀번호 입력
  3. 접속 완료
