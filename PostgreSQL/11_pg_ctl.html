PostgreSQL 데이터베이스 관리

pg_ctl

  • pg_ctl 은 PostgreSQL 데이터베이스를 초기화하거나 start, stop, restart 하는 프로그램이다.
  • pg_ctl 은 이미 운영중인 시스템에서 새로운 데이터베이스를 추가할 때 사용할 수 있다.

pg_ctl 로 신규 데이터베이스 추가

  1. db 초기화 명령어
    pg_ctl initdb -D C:\temp\data2
  2. postgresql.conf 파일에서 port 변경

  3. postgres start
    pg_ctl start -w -D C:\temp\data2
  4. postgres stop
    pg_ctl stop -D C:\temp\data2
  5. service 등록
    pg_ctl register -N postgres-9.5-second -D C:\temp\data2 -w
  6. service 시작
    시작 > 실행 > services.msc > postgres-9.5-second 우측 클릭 > 시작

  7. postgres 접속 (username, password 는 현재 로그인한 계정이 됨)
  8. service 중지
    시작 > 실행 > services.msc > postgres-9.5-second 우측 클릭 > 중지

  9. service 제거
    pg_ctl unregister -N postgres-9.5-second
